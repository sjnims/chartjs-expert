Interactive Legend Examples

1. Linked Dataset Visibility

Clicking first two legend items toggles both datasets together.

Click a legend item to see the behavior.

2. Single Selection Mode

Only one dataset can be visible at a time. Clicking shows only that dataset.

3. Protected Datasets

The "Baseline" dataset cannot be hidden. Other datasets toggle normally.

Try hiding the Baseline dataset - it won't work!

4. HTML Legend with Values

Custom HTML legend showing current values. Click to toggle visibility.

5. Hover Highlighting

Hovering over a legend item highlights that dataset and dims others.
