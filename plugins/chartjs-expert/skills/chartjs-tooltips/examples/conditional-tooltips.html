Conditional Tooltips Examples

Demonstrations of filtering, sorting, and conditionally showing tooltips.

1. Filter: Show Only Values Above Threshold

Tooltips only appear for values greater than 50.

2. Sort: Order Items by Value (Descending)

Tooltip items are sorted from highest to lowest value.

3. Conditional Labels: Style Based on Value

Values above 70 show in green, below 30 in red, others in default color.
