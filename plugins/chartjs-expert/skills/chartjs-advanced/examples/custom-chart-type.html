Custom Chart Type: DerivedBubble

How It Works

This example extends BubbleController to create a custom chart type. The DerivedBubble controller draws a colored box around the first data point, demonstrating how to add custom decorations to existing chart types.

Key Concepts

1. Extend a built-in controller (e.g., BubbleController)
2. Override draw() to add custom rendering
3. Set static id for the chart type name
4. Set static defaults for custom options
5. Register with Chart.register()
